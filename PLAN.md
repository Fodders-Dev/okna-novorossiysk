# PLAN: Лендинг «Окна и двери в Новороссийске»

## Обзор

Разработка разбита на 5 фаз. Каждая фаза содержит конкретные задачи, которые можно выполнять последовательно.

---

## Фаза 1: Инициализация проекта

### 1.1 Инициализация Vite
- Создать проект через `npm create vite@latest ./ -- --template vanilla`
- Очистить шаблонные файлы
- Настроить структуру папок

### 1.2 Структура папок
```
/
├── index.html
├── src/
│   ├── main.js
│   ├── styles/
│   │   ├── index.css        # точка входа стилей
│   │   ├── variables.css    # CSS-переменные
│   │   ├── reset.css        # сброс стилей
│   │   ├── typography.css   # типографика
│   │   └── components/      # стили компонентов
│   └── assets/
│       ├── images/
│       └── icons/
├── public/
│   └── favicon.svg
├── SPEC.md
├── PLAN.md
└── package.json
```

### 1.3 Базовые файлы
- Настроить `index.html` с базовой структурой и мета-тегами
- Создать CSS-переменные в `variables.css`
- Подключить Google Fonts (Inter)

---

## Фаза 2: Базовая верстка

### 2.1 Header
- Разметка: logo, nav, phone, CTA button
- Стили: fixed/sticky, responsive
- JS: бургер-меню для мобильных
- JS: изменение стиля при скролле

### 2.2 Hero-блок
- H1 с основным оффером
- Подзаголовок
- Список преимуществ (3-4 буллита)
- Две CTA-кнопки
- Место под фото мастера
- Адаптивная сетка (2 колонки → 1)

### 2.3 Footer
- Копирайт
- Ссылки
- Контакты

---

## Фаза 3: Основные блоки

### 3.1 Блок «Услуги»
- 6 карточек услуг
- CSS Grid: 3 колонки на десктопе, 1 на мобильном
- SVG-иконки для каждой услуги

### 3.2 Блок «Преимущества»
- 4-6 пунктов
- Иконки + текст
- Flexbox/Grid layout

### 3.3 Блок «Портфолио»
- Галерея изображений
- CSS Grid для сетки
- Lightbox на JS (простой модал)

### 3.4 Блок «Этапы работы»
- 4-5 шагов
- Визуальная связь между шагами (линия/стрелки)
- Номера или иконки

### 3.5 Блок «Цены»
- Таблица или карточки
- Responsive: таблица → карточки на мобильном
- Дисклеймер

### 3.6 Блок «Отзывы»
- Карточки отзывов
- Возможно: слайдер на мобильном (опционально)

### 3.7 Блок «FAQ»
- Аккордеон на JS
- Доступность: ARIA-атрибуты
- Плавная анимация раскрытия

### 3.8 Блок «Контакты»
- Форма с валидацией
- Контактные данные
- Яндекс.Карта (iframe или API)

---

## Фаза 4: Функциональность

### 4.1 Навигация
- Плавная прокрутка к якорям (scroll-behavior или JS)
- Активный пункт меню при скролле (Intersection Observer)

### 4.2 Бургер-меню
- Полноэкранное меню на мобильных
- Анимация открытия/закрытия
- Закрытие при клике на пункт или оверлей

### 4.3 Форма заявки
- Валидация полей (имя, телефон)
- Маска для телефона (опционально)
- Обработка отправки (заглушка или реальный backend)
- Сообщение об успехе

### 4.4 Lightbox для портфолио
- Модальное окно с изображением
- Закрытие по клику на оверлей / Esc
- Навигация между фото (стрелки)

### 4.5 FAQ аккордеон
- Раскрытие/сворачивание по клику
- Только один открытый ответ (опционально)

---

## Фаза 5: SEO и оптимизация

### 5.1 Meta-теги
- title, description
- Open Graph (og:title, og:description, og:image)
- favicon

### 5.2 Schema.org
- JSON-LD разметка LocalBusiness
- Контактная информация
- Услуги

### 5.3 Производительность
- Оптимизация изображений (WebP)
- Lazy loading (loading="lazy")
- Минификация при сборке (Vite делает автоматически)

### 5.4 Доступность
- Контрастность текста
- Фокус-состояния для интерактивных элементов
- ARIA-атрибуты где необходимо

### 5.5 Тестирование
- Проверка на разных устройствах (320px - 1440px)
- Lighthouse аудит
- Проверка форм и навигации
